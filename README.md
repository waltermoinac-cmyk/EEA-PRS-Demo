# EEA-PRS-Demo
Demo funcional en PyTorch del modelo de recomendación centrado en el usuario (EEA-PRS), basado en el paper "User-centric recommendations on energy-efficient appliances in smart grids: A multi-task learning approach". Incluye entrenamiento, evaluación y generación de recomendaciones.
## 🛠 Requisitos

- Python 3.9+
- PyTorch
- NumPy, Pandas, Scikit-learn, Matplotlib

Se recomienda abrir el notebook directamente en **Google Colab**.

## ▶️ Ejecución rápida en Colab

1. Abre este repositorio en GitHub.
2. Haz clic en `EEA_PRS_demo.ipynb`.
3. Pulsa el botón **Open in Colab** (o copia el enlace a Colab).
4. Ejecuta todas las celdas en orden.
5. Prueba:
   ```python
   run_demo(user_id=5)
   run_demo(user_id=120)
